# Prerequisite

## cmake

* windows :

https://cmake.org/download/

* linux : 
```
sudo apt-get install cmake
```

* MacOs - if not still available with Xcode install : 
```
brew install cmake
```

# Installation

```
npm install nodeopencv
```

# Usage

```
import {opencv} from 'nodeopencv'
```
